HTML


Grundstruktur

  • <!DOCTYPE html>: Deklariert den Dokumenttyp und die HTML-Version.

  • <html>: Das Wurzelelement, das den gesamten HTML-Code umschließt.

  • <head>: Enthält Metadaten, Skripte, Links zu CSS-Dateien und den Titel der Seite.

    • <meta charset="UTF-8">: Legt die Zeichenkodierung der Seite fest.

    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Sorgt für eine responsive Gestaltung auf mobilen Geräten.

    • <title>: Definiert den Titel der Webseite, der im Browser-Tab angezeigt wird.

    • <link rel="stylesheet" href="style.css">: Verknüpft eine externe CSS-Datei für die Gestaltung der Webseite.

  • <body>: Enthält den sichtbaren Inhalt der Webseite.

Inhaltsstruktur


  • <header>: Definiert den Kopfbereich der Seite.

  • <main>: Enthält den Hauptinhalt der Seite.

  • <footer>: Definiert den Fußbereich der Seite.

  • <section>: Definiert einen Abschnitt innerhalb des Hauptinhalts.

  • <article>: Definiert einen eigenständigen Artikel oder Beitrag.

  • <nav>: Definiert eine Navigationsleiste.

  • <aside>: Definiert Inhalte, die indirekt mit dem Hauptinhalt verbunden sind, wie z.B. Seitenleisten.

  • <div>: Ein generisches Container-Element, das zur Gruppierung von Inhalten verwendet wird.

  • <span>: Ein generisches Inline-Element, das zur Gruppierung von Text oder anderen Inline-Elementen verwendet wird.
