# HTML CSS JS Cheatsheet

Eine praktische Referenz-Seite mit den wichtigsten HTML-, CSS- und JavaScript-Befehlen – alles auf einer Seite zum schnellen Nachschlagen beim Coden. Es ist auch ein simpler, aber nützlicher Guide für Anfänger – gebaut, um selbst besser zu lernen und anderen zu helfen.

Eine lange, übersichtliche Cheatsheet-Seite mit 1800+ Zeilen Code: Alle wichtigen Tags, Properties und Methoden mit kurzen Erklärungen. Cleanes Design, schwarz-weiß, mit blurry Hintergründen bei Gruppen und Hover-Effekten und etwa 80%-90% von allen Tags/Code – perfekt zum schnellen Suchen ohne Google-Stress. Noch basic, aber ich war noch nie so glücklich, dass ich mich für Hilfe zu Anfängern gesetzt habe! 📚✨

### Technologies
- HTML
- CSS

Alles vanilla – kein Framework, kein JavaScript im Code (nur als Referenz-Inhalt), direkt in VS Code gebaut und auf Netlify deployed.

### Features
- Anchor-Navigation oben (großer Titel + Links zu HTML/CSS/JS springen direkt zum Abschnitt)
- Responsive – gut lesbar auf Handy und Desktop
- Hover-Effekte (z. B. bei Links oder Elementen)
- Blurry Hintergründe bei jeder Gruppe für bessere Übersicht
- Einfaches Kopieren: Alle Beispiele und Erklärungen lassen sich easy markieren und kopieren
- Footer-Credit: **[Designed by Omar](mailto:omar.aly17@outlook.de)**
- Sehr übersichtlich und einfach zu bedienen – Fokus auf Inhalt, keine unnötigen Extras

### The Process
Ich wollte Anfängern eine wirkliche Hilfe geben, damit sie nicht ständig denken müssen „och man, das ist nicht das was ich brauche“ oder „ich muss die ganze Zeit suchen“. Deshalb habe ich alles Wichtige in einer einzigen Seite gesammelt.  

Am schwierigsten war, alles übersichtlich zu behalten und die IDs bei so viel Code (1800+ Zeilen) richtig zu verbinden. Dabei habe ich gelernt: Anchor-Links richtig setzen und blurry Hintergründe für Gruppen zu stylen.  

Ich war noch nie so glücklich, dass ich mich für Hilfe zu Anfängern gesetzt habe – nächstes Mal vielleicht mehr Features, aber das hier ist schon ein cooler Schritt!
